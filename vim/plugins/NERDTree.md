# [NERDTree](https://vimawesome.com/plugin/nerdtree-red)

Plugin para mostrar un tree de navegación de archivos

## Comandos

| Comando    | Descripción                |
| ---        | ---                        |
| `<C-n>`    | Mueve el cursor a NerdTree |
| `<C-t>`    | Cierra NerdTree            |
| `<C-f>`    | Buscador en NerdTree       |

> La letra `C` significa `Ctrl`
